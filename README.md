1. 不能上传无码的不能看的部位，血腥，或非常 恶 趣 味 的东西！适量的讨论

5. 请勿外传
6. 如阅读后因为不遵守规则发生任何事情，站主不负责任何责任
7. 非管理员的允许下，严禁修改readme.md，以防出现乱改的情况
